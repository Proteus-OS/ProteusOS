## 🛡️ ProteusOS - Distribución SOC

Bienvenido al repositorio oficial de descarga de **ProteusOS**. Esta distribución ha sido diseñada para proporcionar un entorno listo para el análisis de seguridad y operaciones SOC sobre **Debian 13**.

## 📥 Descargas
Las imágenes ISO oficiales se encuentran en la sección de [Releases](https://github.com/Proteus-OS/ProteusOS/releases).

## 🚀 Inicio Rápido
1. Descarga la imagen ISO (`v1.0-Aegis`).
2. Graba la imagen en un pendrive (recomendado: **Ventoy** o `dd`).
3. Inicia tu hardware o VM desde el medio USB.
4. Selecciona tu perfil de instalación (**LITE**, **STANDARD** o **FULL**) desde el menú TUI.
